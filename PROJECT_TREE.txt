Key Files to Know:
==================

1. start_bot.py
   - Quick start script
   - Run this to start the bot
   - Command: python start_bot.py

2. bot/bot.py
   - Main bot application
   - Registers all handlers
   - Entry point for bot

3. bot/handlers/
   - user_handlers.py: Registration, rules, support
   - wallet_handlers.py: Balance, deposits, withdrawals
   - game_handlers.py: Game logic, card selection

4. */models.py
   - Database schema definitions
   - User, Wallet, Game, BingoCard, Transaction

5. */admin.py
   - Django admin interface
   - Manage users, approve transactions

6. .env
   - Configuration (BOT_TOKEN, etc.)
   - MUST be configured before running

7. requirements.txt
   - Python package dependencies
   - Install with: pip install -r requirements.txt

Database Tables:
================

users           - User profiles
wallets         - User balances (main + bonus)
transactions    - All financial transactions
deposits        - Deposit details & proofs
withdrawals     - Withdrawal requests
games           - Game instances & state
bingo_cards     - Player cards & grids

Quick Commands:
===============

# Start bot
python start_bot.py

# Django admin
python manage.py createsuperuser
python manage.py runserver

# Database
python manage.py makemigrations
python manage.py migrate

# Install dependencies
pip install -r requirements.txt
